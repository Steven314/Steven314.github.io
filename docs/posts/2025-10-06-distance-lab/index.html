Distance to City Center

toledo_lat <- 41.652778
toledo_lng <- -83.537778

Wikipedia gives the center of Toledo as (41.652778, -83.537778).

Letâ€™s establish a few distance metrics and compare the results.

euclidean <- function(ref, new) {
  sqrt(rowSums((ref - new)^2)) *
    (pi / 180) * # convert to radians
    6371 # km, the radius of the earth
}

manhattan <- function(ref, new) {
  rowSums(
    abs(ref - new) * 
      (pi / 180) *
      6371
  )
}

# and also Haversine from `{geosphere}`

So now we can calculate the distance to the city center with Euclidean, Manhattan, and Haversine distances.

house_sf_dist <- house_sf |> 
  mutate(
    longitude = st_coordinates(geometry)[, 1],
    latitude = st_coordinates(geometry)[, 2],
    toledo_lat, 
    toledo_lng,
    # redefine age
    age = as.numeric(as.character(syear)) - yrbuilt
  ) |> 
  mutate(
    distance_euclidean = euclidean(
      cbind(toledo_lat, toledo_lng), 
      cbind(latitude, longitude)
    ),
    distance_manhattan = manhattan(
      cbind(toledo_lat, toledo_lng), 
      cbind(latitude, longitude)
    ),
    distance_haversine = geosphere::distHaversine(
      cbind(toledo_lng, toledo_lat), 
      cbind(longitude, latitude)
    ) / 1000 # convert meters to kilometers
  )

Information about each variable can be found in the associated help file. It can be opened by typing ?house in the console. Although it is not the most thorough explanations of each variable, it is still helpful.

We can make a list of relationships to consider. In these there are plenty of hypotheses.

  • As you move toward the city center, lots get smaller, thus prices go down. The converse also being the case.
  • Having more bedrooms, bathrooms, or stories would increase price.
    • This may be correlated with lot size.
  • The city center may be older and perhaps older houses have lower prices.

All of those are statements we can test.

We can also come up with statements we canâ€™t test with the data currently at hand.

  • Houses close to highways have lower value.
  • Proximity to amenities, such as schools, parks, and entertainment, can contribute to changes in housing prices.
  • A house of poor quality would have lower value.

Linear Model

The lm() function can be used for a linear model. It takes a dataset for training and a formula.

Splitting Data

When training a model to be used for predicting new data you should split into training and testing (and potentially validation) sets. When there is a spatial component you may also want to consider the spatial relationships.

  • Spatial Data Science seems like a good resource for more info on this. I have yet to read it, but it is on the list for the future.
  • Tidy Modeling with R, chapter 5 is also a good resource for information on training a model. I have read this whole book and can recommend it.
  • {spatialsample} is a package in the tidymodels and tidyverse ecosystem that can perform spatial sampling. I havenâ€™t used it yet, but likely will within the next year.

Why?

Other factors have spatial variance. Distance isnâ€™t the only thing changing between observations. We have an endogeneity issue or omitted variable bias. That should sound familiar from an econometrics class. If not, you should be covering that soon.

This means that some of the variation in price is better explained by the other variables we included.

Model Conclusion

If we are happy with the reasoning for our model (i.e. we donâ€™t expect any omitted variable bias or other statistical issues), we could attempt to claim a causal effect of distance on housing prices.

The predictions can be found with the predict() function.

Notice how we have to use exp() wrapping predict. This is because we used log(price). We need to reverse the logarithm.

More Complicated Models

The actual model technique may not necessarily be more complicated, but the feature engineering (TMwR chapter 8) is fancier.

  • If a house in the same neighborhood (or at least nearby) sold the previous year, the housing prices in the neighborhood may be influenced. The proximity with a recently sold house and how similar (also distance, but in non-spatial dimensions) could affect how strong that influence is.
    • I couldnâ€™t find a paper that covered this, but I didnâ€™t look for very long.
  • Proximity to other points-of-interest, such as schools, parks, entertainment, or highway access, can be considered.
    • Measuring Highway Impacts on House Prices Using Spatial Regression
  • Using travel time instead of straight line distance.
